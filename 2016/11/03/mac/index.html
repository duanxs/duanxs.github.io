1 开始安装

1.0 制作 mac 启动盘

1.0.1 制作 mac 分区

按 win ＋ X 键调出菜单，打开磁盘管理 ，压缩出合适空间，并按如图操作：

1.0.2制作 mac 启动安装盘.准备

右键以管理员身份打开 TransMac

选择插入的U盘，右击，按下图操作，一路确定

1.0.3制作 mac 启动安装盘.写入

选择mac镜像，并点ok

等待写入结束

至此，启动盘制作完毕

1.1 开始安装

1.1.0

重启电脑，华硕应该是Esc键，开机按住不放直到出现下图窗口

图1.1.1

选择 UEFI：你的U盘，回车，会进入下图：

图1.1.2

光标移到：Boot OS X Install form Install OS X El Capitan，然后按空格 ，进入下图：

图1.1.3

选择 Boot xxxxx with inject kexts，回车，如若卡在下图，AMFILoadTrusted，重启

图1.1.4

进入clover界面

图1.1.5

光标移动到Option选项，空格

进入 PCI devices菜单

在 第一个选项上 回车，然后返回.重复 图1.1.2 操作

图1.1.6

代码跑完，进入上图界面

选择 磁盘工具

选择分区

选择给mac分的盘，起个名，然后点应用

分好区后，选择mac盘，点上面抹掉按钮，抹掉。

退出磁盘工具

选择 安装 OS X ，出现上图，选择mac盘，点安装，等待就OK了
